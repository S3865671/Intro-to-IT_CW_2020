Project Idea

Overview

The idea for this project is an automated pet minder system customised around the humble Chicken.Backyard chickens can be a little more challenging should you wish to head away for a weekend. With my project outcome shall give Backyard Chicken owners some peace of mind that the birds that provide food are looked after. With many different products on the market for Cat and Dog owners such as home monitoring and microchipping why should the backyard chicken which are on the rise as the fourth most popular backyard pet in Australia.With Backyard Chicken ownership on the rise including an increase on demand from Covid 19 let us focus on using technology to simply looking after the Chickens welfare.

Motivation

The saying the early bird catches the worm is very true therefore during the summer months in Brisbane Australia daybreak is very early and therefor the backyard chicken awakens. As keeper of backyard chicken’s myself they can get very restless as all they want is to escape the coup to free range. There is already a solution with an automated door opener set on a timer however consider for a moment the door opens because the timer was set to early. Income the cat or the dog that have escaped their yard on the hunt for a playmate.

Description

Automated pet minder system is a project about backyard chicken welfare management system. By connecting the door opener to the Internet via WIFI and programming its location along with the sunrise this in turn will mitigate the risk of predators in the morning. Fast forward time to bedtime can we apply the same principle as the morning setting sure although could we also use technology with NFC tags and receiver to count the returning flock after a predetermined time then the door closes. Monitoring inside the interior and exterior can be done using a camera system. Using the exterior camera as surveillance for predators with a speaker system to scare the intruder. Once we have the control unit connected to the Internet there are a number of integrated systems that could be developed or addition of current technologies. One such system would anytime monitor via a smart phone application. With welfare at the forefront of this project the supply of food and water need to be considered. Unlike humans, chickens do not overeat and tend to graze all day according to their needs (Brasch,2007). For this part of the project it will be low tech with the use of a chicken feeder that is topped up as required. Now we need to consider the programming with a simple user interface both from a browser mobile app considering what method does the pet owner wish to deploy for the door opener. Option 1. Is a set and forget timer. Option 2. A sunlight activated trigger or Option 3. Open time set using internet weather reporting. Camera control would also need to be programmed for Interior camera set to off automatically during the day to allow for maximum solar power input to the battery, alternatively the pet owner will have the ability to active cameras remotely via the app. For the exterior cameras motion detectors that will not only trigger the camera but also alert the owner. With many different peripherals to be taken into consideration the project is a scalable solution based on the pet owners needs.

Tools and Technology

Using a raspberry pi connected to internet as the processor the project then will require additional peripherals. The Project fore mention shall incorporate firstly a power source allowing for the possibility of a remote location somewhere in the back yard. Added to the processor will be both interior and exterior cameras including motion detector sensor. As for the predator deterrent device there are many ideas ranging from a speaker the pet owner can scare the offender thru to using an automated sprinkler system.

Skills Required

Most of the tech is already available so the skills required will be more about programming the processer and using a web page and app for control. The programming would need to consider inputs from the internet coupled with triggers from sensors and required action. The web and app would allow human activation as required should an alert come up. As there are many smartphone programming platforms this would be one of final programming tasks undertaken.

Outcome

Will this solution be appropriate for every backyard chicken owner ?

In short possible not, however as technology progresses does a project like this dose not need to be just for backyard chickens. Currently there are many products on the market for home security and baby monitoring. Even if this is not commercially viable it shall provide myself with some peace of mind based on my circumstance.
